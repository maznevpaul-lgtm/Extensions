Полезные расширения для вашего браузера

Скачивайте наши кастомные плагины напрямую. Безопасно, быстро и с открытым исходным кодом.

Из-за политики безопасности браузеров прямая установка со сторонних сайтов заблокирована. Требуется ручная установка файла, но обновления будут прилетать автоматически!

  1. Скачайте файл расширения (в формате .crx) по кнопке ниже.
  2. Откройте новую вкладку в браузере и перейдите по адресу chrome://extensions/ (или edge://extensions/).
  3. В правом верхнем углу обязательно включите "Режим разработчика" (Developer mode).
  4. Перетащите скачанный файл .crx из папки "Загрузки" прямо в окно браузера с открытой страницей расширений.
  5. Подтвердите установку. Готово! В дальнейшем расширение будет обновляться само, когда автор загрузит новую версию на сайт.

Для Яндекс Браузера и Opera: При нажатии на скачивание браузер может сразу начать установку. Сверху появится сообщение — нажмите на него, чтобы перейти в раздел расширений и разрешить установку там.

⚠️ Важно: Если вы разархивируете скачанный .crx файл и установите его папкой, автоматические обновления прилетать НЕ БУДУТ!

О мобильных браузерах: Не все мобильные браузеры поддерживают расширения. Рекомендуемые варианты:
  • Kiwi Browser — позволяет устанавливать почти любые расширения из Chrome Web Store.
  • Yandex Browser — поддерживает часть расширений для Chrome и Opera.
  • Lemur Browser — специализированный браузер с упором на поддержку расширений.

Браузер отключает расширение после установки?

Современные браузеры могут автоматически отключать расширения не из официального магазина. Чтобы браузер доверял вашему плагину, добавьте его в "Белый список".

💡 Обратите внимание: В эти скрипты уже внесены ID трёх наших расширений! Если вам нужны только они, просто запустите скрипт и на первый же запрос нажмите Enter, ничего не вводя.

Как добавить любые другие расширения: Найдите расширение на странице chrome://extensions/ и скопируйте его ID (32 буквы).
  • 1. Появляется запрос: Введите ID расширения #1: — вставляете ID, жмёте Enter.
  • 2. Сразу появляется запрос #2 — так можно добавить любое количество ваших расширений.
  • 3. Когда запросит следующий ID, а вам больше ничего не нужно добавлять — просто ничего не пишите и нажмите Enter в пустой строке.
  • 4. Обязательно перезапустите браузер (закройте все окна), чтобы настройки применились.
Скачать скрипт .bat (Windows)

Нажмите на файл правой кнопкой мыши ➔ Запуск от имени администратора.

Скачать скрипт .sh (Mac)

Откройте Терминал, напишите bash , перетащите файл в окно и нажмите Enter.
